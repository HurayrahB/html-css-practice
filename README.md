# HTML/CSS/JS Practice

Working through guided projects from [Dash](https://dash.generalassemb.ly/) by General Assembly.

## Topics Covered

- HTML structure and semantic elements
- CSS styling, layout, and responsive design with media queries
- CSS animations and keyframes
- jQuery for DOM manipulation and event handling
- Vanilla JavaScript for interactivity

## Practice Projects

| File | Description |
| ------ | ------------- |
| `personal-web.html` | Simple personal landing page with a background image and email input |
| `personal-blog.html` | Multi-article blog layout with a responsive header and like buttons |
| `small-business.html` | Restaurant menu page with click-to-expand dish descriptions |
| `css-robot.html` | CSS-drawn robot with laser eye toggle, random background color, and a dancing gif |
| `js-traffic-light.html` | Interactive traffic light controlled by stop/slow/go buttons |
| `madlib-game.html` | Mad Libs game that collects user inputs and assembles them into a story |
